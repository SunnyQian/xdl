1. 什么是变量
	存储值的容器，变量用于计算或者是其他操作
2. 显式的变量类型有几种，分别是什么
	以typeof函数返回的变量类型为判断标准
	bsofun 如此有趣
		Boolean		布尔值
			true
			false
		String 		字符串
		Object		对象
		Function 	函数
		Undefined	未定义
			1. 变量声明，未赋值
			2. 显式的将undefined值赋值给变量

		Number		数值
			整数
			浮点数
			NaN	是一个不是数字的数值类型


3. 变量名的命名规范
	1. 以字母、下划线_、$开头
	2. 后跟任意长度的数字、字母、下划线、$
	3. 严格区分大小写
	4. 变量名命名尽量有意义
	5. 推荐使用驼峰命名法
	6. 不能使用系统的关键字和保留字
		关键字：
			系统已经占用的，有特殊意义的！
			var/typeof 
		保留字：
			现在我不用，但是我打算以后用，你不能用！
