## jQuery 的功能和优势
1. 简化 javascript代码操作	
2. 不需要再来关心兼容性的问题 
3. 批量操作 DOM
4. 像css 一样来访问和操作 dom
5. 封装一些动画
6. 大量 JQuery 插件
7. 让 Ajax 技术更加完美  


## jQuery 的设计思想
```
	模拟 css 的选择器
	jQuery 方法函数化	
	链式操作	
	取值赋值合体	

```

## load  和   ready  区别
```
load  要等待网页全部加载完毕再执行(包括图片)
		只能一次

ready  只需要 网页中 dom 结构加载完毕就可以执行
		可以使用多次

```
